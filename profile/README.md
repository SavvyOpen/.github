**Chung Lim Lee**

Many things can be achieved simply with the right idea.

* SO ChatX – a fast, secure, real-time web chat built from scratch using HTML, CSS, JavaScript, PHP, and MySQL.  Demonstrates advanced features that typical frameworks do not provide by default, such as live SSE (Server-Sent Events) messaging, MFA (Multi-Factor Authentication) with TOTP, portable chat history, incremental updates, single-session control, and group chats.

* SO Gloper – A unified React library that combines state, logic, and dataflow in a universal namespace system with dual modes (quick setup vs maximum IDE support), dynamic state creation, programmatic API, built-in persistence, lazy loading, auto-immutability, and React 18+ concurrency—letting you build scalable, maintainable apps faster with minimal boilerplate.

* SO Othello – the world’s first Othello app with an unlimited endgame generator, perfect-play solver, interactive play, effortless social sharing, and official recognition by worldothello.org.
